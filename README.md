# ABCENTRUM HTML email project

Готовый проект HTML-письма для ABCENTRUM с главным фокусом на объекте «Промтехнопарк в Митино» и доступных площадях в аренду.

## Состав проекта

- `email.html` — готовое HTML-письмо для локального просмотра с относительными путями к изображениям.
- `index.html` — копия `email.html` для быстрого открытия в браузере.
- `email-cdn-template.html` — версия для рассылочного сервиса: замените `{{ASSET_BASE_URL}}` на HTTPS-папку с изображениями.
- `plain-text.txt` — текстовая версия письма.
- `subject-preheader.txt` — тема письма и прехедер.
- `assets/` — оптимизированные изображения для письма.
- `source/` — исходные материалы, загруженные для проекта.
- `docs/technical-spec.md` — итоговое ТЗ.
- `docs/qa-checklist.md` — чек-лист приемки и тестирования.
- `CODEX_SEND_EMAIL_PROMPT.md` — инструкция для Codex по отправке рассылки.
- `.env.example` — шаблон переменных окружения для Mail.ru SMTP.

## Перед отправкой

1. Загрузите файлы из `assets/` на HTTPS-хостинг или в рассылочный сервис.
2. В `email-cdn-template.html` замените `{{ASSET_BASE_URL}}` на абсолютный URL папки с изображениями.
3. Подставьте реальную ссылку отписки вместо `{{unsubscribe_url}}`.
4. Отправьте тесты в Gmail, Apple Mail, Outlook, Mail.ru и Яндекс Почту.

## Техническая база верстки

Письмо сверстано в подходе профессиональной email-разработки: HTML5 doctype, `lang="ru"`, скрытый прехедер, табличная структура, inline CSS, media queries для мобильной адаптации, VML-кнопки для Outlook, alt-тексты изображений, фиксированный контейнер 640 px с адаптацией на мобильных экранах.

## Отправка через Codex

Для запуска рассылки используйте инструкцию `CODEX_SEND_EMAIL_PROMPT.md`. В проекте уже зафиксированы правила: `To` всегда только `s.zharov@abcentrum.ru`, все получатели идут только в `Bcc`, `Cc` не используется.

Секреты не храните в репозитории. Создайте `.env` на основе `.env.example` или передайте переменные окружения Codex. Реальный `SMTP_PASS` не должен попадать в markdown, HTML, JS, JSON, логи или git.
