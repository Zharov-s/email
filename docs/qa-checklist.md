# QA-чеклист HTML-письма ABCENTRUM

## Контент

- [ ] Первый экран посвящен только Митино.
- [ ] Есть фраза «ввод в эксплуатацию — сентябрь 2026 года».
- [ ] Есть факты: 11 776 м², класс A+, 1,5 МВт, ул. Барышиха, 37А.
- [ ] Некрасовка и Грекова размещены ниже и визуально не конкурируют с Митино.
- [ ] Telegram-блок содержит `@ABCENTRUM_DEV`.
- [ ] Нет длинных пояснительных абзацев и лишней воды.

## Ссылки

- [ ] «Скачать презентацию» открывает Google Drive в браузере.
- [ ] «Смотреть объект» / «Смотреть объекты» ведет на https://abcport.ru/.
- [ ] «Подписаться на Telegram» ведет на https://t.me/abcentrum_dev.
- [ ] Подставлена реальная ссылка отписки вместо `{{unsubscribe_url}}`.

## Изображения

- [ ] Все изображения открываются с HTTPS-адресов после загрузки на хостинг.
- [ ] Hero-изображение — фасад Митино.
- [ ] Использованы фасады Некрасовки и Грекова.
- [ ] Использован логотип ABCENTRUM.
- [ ] Использован QR / визуал Telegram-постера.
- [ ] У всех изображений есть alt-тексты.
